# jTask

### Requirements
* [Java](https://openjdk.org)  
* [Maven](https://maven.apache.org)  
* [Spring](https://spring.io)
* [JUnit](https://junit.org)
* [PostgreSQL](https://www.postgresql.org)

#### Unit tests
* Windows, MacOS/Linux
```
mvn clean test
```
#### Build application
* Windows, MacOS/Linux
```
mvn clean package
```
#### Run application
* Windows, MacOS/Linux
```
java -jar jtask-x.x.x.jar
```
