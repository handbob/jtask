# jTask

### Requirements
* [Java](https://openjdk.org/)  
* [Maven](https://maven.apache.org)  
* [Spring](https://spring.io)

#### Build and run
* Windows, MacOS/Linux
```
mvn spring-boot:run
```
#### Build without run
* Windows, MacOS/Linux
```
mvn install clean package
```
#### Run
* Windows, MacOS/Linux
```
java -jar jtask-x.x.x.jar
```
