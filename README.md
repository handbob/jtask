# Spring RESTful API web services

### Requirements
[Java](https://adoptium.net), [Gradle](https://gradle.org), [Spring](https://spring.io)

### Build & Run project

#### Run
* Windows, MacOS/Linux
```
gradle bootRun
```
#### Build
* Windows, MacOS/Linux
```
gradle build
```
#### Run Jar file
* Windows, MacOS/Linux
```
java -jar todolist-1.0.0.jar
```
