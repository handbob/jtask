# Web RESTful APIs in Spring

### Requirements
[Java](https://adoptium.net), [Gradle](https://gradle.org), [Spring](https://spring.io)

### Build & Run

#### Build
* Windows
```
.\gradlew.bat build
```
* MacOS/Linux
```
./gradlew build
```
#### Run Jar file
* Windows, MacOS/Linux
```
java -jar todolist-1.0.0.jar
```
