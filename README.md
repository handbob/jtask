# Spring RESTful API web services

### Requirements
[Java](https://adoptium.net), [Maven](https://maven.apache.org), [Spring](https://spring.io)

### Build & Run project
#### Build
* Windows, MacOS/Linux
```
mvn package
```
#### Run Jar file
* Windows, MacOS/Linux
```
java -jar todolist-1.0.0.jar
```
